Instance Age Filter

The instance age filter allows for filtering the set of EC2 instances by their LaunchTime, i.e. all instances older than 60 or 90 days. The default date value is 60 days if otherwise unspecified. Configuring a specific value for instance-age to report all instances older than 90 days.

policies:
  - name: old-instances
    resource: ec2
    filters:
      type: instance-age
      days: 90
